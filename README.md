- **Deployment:** Docker, AWS  

## 🚀 Installation  
Clone the repository and set up the environment:  

```sh
git clone https://github.com/axay28/Radiant.git  
cd Radiant  
pip install -r requirements.txt  
```

Run the server:  
```sh
python app.py  
```
